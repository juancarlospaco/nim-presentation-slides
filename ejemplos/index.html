Iterators

Iteradores (Iterators), tambien llamado Iteradores-enlinea (Inline-Iterators).

Inline-Iterators son los Iteradores por defecto en Nim,

son compilados a Loops optimizados de alta performance.

Un Iterador puede ser tambien un Generic, como es iterator_generic.nim

En el ejemplo iterator.nim, un Iterator toma un string invierte el orden de las letras.

nim c -r iterator.nim

nim c -r iterator_generic.nim

Crosscompile custom Esto es 100% Opcional.
Crosscompile usando:
-d:crosswin
En lugar de:
--gcc.exe:/usr/bin/x86_64-w64-mingw32-gcc --gcc.linkerexe:/usr/bin/x86_64-w64-mingw32-gcc

Ejemplos

Antes: nim c --cpu:amd64 --os:windows --gcc.exe:/usr/bin/x86_64-w64-mingw32-gcc --gcc.linkerexe:/usr/bin/x86_64-w64-mingw32-gcc hello.nim

Despues: nim c -d:crosswin hello.nim

La ruta del *.CFG es (Crearlo si no existe):

~/.config/nim.cfg
